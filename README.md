# Tableau de Bord - Matières Premières Agro-alimentaires

## 📊 Description

Application web complète pour le suivi en temps réel des prix des matières premières agricoles et alimentaires. Conçue pour les acheteurs professionnels du secteur agro-alimentaire.

## ✨ Fonctionnalités

- **Suivi des Prix**: 12 matières premières clés (céréales, oléagineux, sucres, huiles, produits laitiers, viandes)
- **Graphiques d'Évolution**: Historique sur 30 jours avec Chart.js
- **Actualités du Marché**: Flux d'actualités pertinentes
- **Analyse de Marché**: Sentiment du marché et recommandations
- **Signaux d'Achat**: Détection automatique d'opportunités
- **Alertes Prix**: Notifications pour variations importantes
- **Filtres**: Sélection/désélection par produit
- **Export PDF**: Génération de rapports PDF
- **Export Excel**: Export des données en format Excel
- **Auto-actualisation**: Actualisation automatique toutes les 5 minutes
- **Responsive Design**: Compatible mobile, tablette et desktop

## 🚀 Utilisation

1. **Ouvrir** `index.html` dans un navigateur moderne
2. **Filtrer** les produits en cliquant sur les chips de filtre
3. **Actualiser** manuellement ou activer l'auto-actualisation
4. **Exporter** les données en PDF ou Excel selon vos besoins

## 🔄 Actualisation des Données

- **Manuelle**: Cliquer sur le bouton "🔄 Actualiser"
- **Automatique**: Cocher "Auto (5 min)" pour actualisation toutes les 5 minutes
